## Movie Project

A movie rental application implementing a system that that allows different types of users to rent movies and handle payments.

## Running the Program
1. Download the project's ZIP file.
2. Run the program under `Main.java`.
---

## Source Files Location
All source files are located in the following directory:
- `/src`

### Source Files:
- `/src/ExternalMember.java`
- `/src/Main.java`
- `/src/Movie.java`
- `/src/MovieRental.java`
- `/src/Payment.java`
- `/src/Person.java`
- `/src/Student.java`
